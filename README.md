# prismagram

Intagram clone with Express/Prisma/React

## User Stories

- [x] Create an account
- [x] Request Secret
- [x] Confirm Secret (Login user)
- [x] Like / Unlike a photo
- [x] Comment a photo
- [x] Search by user
- [x] Search by location
- [ ] Follow a user
- [ ] Unfollow a User
- [ ] See user profile
- [ ] See the full photo
- [ ] Upload photos
- [ ] Edit the photo (Delete)
- [ ] See the feed
